# 系统管理工具

## htop - 更好用的 top

htop 是一个交互式的系统监控工具，相比传统 `top` 命令更易用、美观。

- 彩色显示 CPU/内存/交换空间使用情况
- 支持进程树视图
- 可以直接杀掉进程
- 支持排序和过滤
- 支持鼠标操作

```sh
brew install htop
```



## btop - 高颜值的系统监控（推荐）

btop（Better TOP）是 htop 的升级版，是一个现代化、彩色、交互式的系统监控工具。

- 美观的图形界面：CPU、内存、网络、磁盘使用可视化
- 实时监控：CPU、内存、磁盘 I/O、网络流量、进程信息
- 交互式操作：支持键盘和鼠标操作
- 可排序和筛选进程
- 跨平台：Linux、macOS、BSD

```sh
brew install btop
```



## ifstat - 网络流量实时监控

ifstat 是一个轻量级网络带宽监控工具，可以实时显示网络接口的流量（上传/下载速度）。

- 简单快速：无需复杂配置
- 实时刷新：每秒更新网络流量
- 支持多网卡：同时监控多个接口
- 轻量级：只输出关键数据

```sh
brew install ifstat
```
