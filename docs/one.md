# 终端效率神器

## fzf - 模糊搜索神器

```shell
brew install fzf
$(brew --prefix)/opt/fzf/install
```
可以快速搜索历史命令、文件路径、任何可搜索的内容。

## ripgrep(rg) - 比 grep 强 100 倍的搜索

速度极快，开发者最常用
```shell
brew install ripgrep
```
## bat - 带语法高亮的 cat
替代 cat，超清晰
```shell
brew install bat
```
## exa 或 eza - 彩色 ls（更现代）
替换 ls
```shell
brew install eza
```
## tldr - 用一句话解释命令用法
比 man 页少废话很多
```shell
brew install tldr
```
